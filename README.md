SAFECross 🚦
Automatic Railway Crossing Gate System using ESP32 and Sensors.

📁 Project Structure
SAFECROSS/ ├── docs/ # Documentation, diagrams, reports ├── hardware/ # ESP32 sender & receiver code + schematics ├── sensors/ # IR & Piezo sensor test code ├── software/ # IoT supporting scripts ├── data/ # Sample datasets / logs ├── models/ # ML models (if any) ├── ppt/ # Presentation slides ├── tests/ # Testing & validation files ├── .github/ # Workflow (CI/CD) ├── LICENSE └── README.md

🧭 Folder Overview
Folder	Description
docs/	Project documentation and diagrams
hardware/	ESP32 firmware and schematics
sensors/	Sensor calibration and testing
software/	Python or IoT scripts
data/	Example sensor data
ppt/	Project presentation files
tests/	Hardware/software testing files
.github/	GitHub workflow automation files
🔧 Technologies Used
Microcontroller: ESP32 DevKit V1
Sensors: IR Sensor, Piezo Sensor
Language: Arduino C++
Tools: Arduino IDE
Communication: Wireless (Sender ↔ Receiver)
🎯 Purpose
The SAFECROSS system automatically controls railway crossing gates using real-time train detection through multiple sensors, preventing accidents and ensuring safe transportation.
